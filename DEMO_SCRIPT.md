# AI Interview Coach — MVP Demo Script (~5 minutes)

---

## [0:00–0:40] Introduction

> Hello everyone. Today I'm presenting my capstone project: **AI-Assisted Job Interview Practice Platform for Mongolia**.
>
> The problem is simple — job interviews are stressful, especially for Mongolian job seekers who have limited access to professional interview coaching. There's no affordable way to practice and get feedback in Mongolian.
>
> My solution is a web-based platform where users can practice mock interviews entirely in Mongolian — they hear questions spoken aloud, respond by voice or text, and receive detailed NLP-based feedback on their answers.

---

## [0:40–1:20] Tech Stack Overview

> Let me briefly cover the tech stack before the demo.
>
> The **backend** is built with FastAPI and Python. It runs a custom NLP pipeline that analyzes answers across five dimensions: word count, filler words, vocabulary richness, answer structure using STAR method detection, and relevance to the selected job category.
>
> For **speech-to-text**, I'm using a fine-tuned Whisper model — specifically `bayartsogt/whisper-large-v2-mn-13` from HuggingFace — which was trained on Mongolian speech data and achieves about 20% word error rate. I converted it to CTranslate2 format with float16 quantization for faster inference on CPU.
>
> For **text-to-speech**, the platform uses Microsoft Edge TTS with a Mongolian voice to read questions aloud, making it feel like a real conversation.
>
> The **frontend** is React with Vite — clean, simple, bilingual in Mongolian and English.

## [3:30–4:30] Results Page

> Now let's look at the **results page**.
>
> At the top, I see my **overall score** out of 100, broken down into five dimensions: word count, filler words, vocabulary, structure, and relevance.
>
> Below that is the **key feedback** section — strengths in green, areas to improve in yellow, and suggestions in blue. This feedback is generated by averaging metrics across all my answers, so it's coherent and doesn't contradict itself.
>
> In the **per-question breakdown**, I can see each question with my answer, the score, and an improvement hint. And if I click **"Жишиг хариулт"** — "Sample Answer" — I can see a model answer to compare with mine.
>
> Finally, I can **export this entire report as a PDF** for my records, or click **Start New Session** to practice again.

## [4:30–5:00] Closing

> To summarize — this platform combines speech-to-text, NLP analysis, and text-to-speech to create an accessible interview coaching experience in Mongolian. The key technical contributions are the fine-tuned Mongolian Whisper integration, the rule-based NLP scoring pipeline, and the bilingual interface.
>
> For future work, I plan to improve STT accuracy — potentially integrating Google Cloud Speech API — add more question categories, and implement a progress dashboard with score trend charts.
>
> Thank you. I'm happy to take any questions.

## Demo Tips

- **Before the demo**: Make sure both backend and frontend are running. Do one test run.
- **If STT is slow**: Mention it's running a 2.9GB model on CPU — production would use GPU.
- **If transcription has errors**: "As you can see, the transcription captures the meaning well, though Mongolian STT is still an active research area with ~20% WER."
- **If asked about scoring**: The NLP pipeline uses rule-based analysis — no LLM calls — so it's fast, free, and runs entirely locally.
